2022.04.29 - 2022.06.29

overview

CU POST는 가까운 편의점에서 택배를 보낼 수 있는 서비스로 어플을 사용하여 택배 예약과 배송 내역을 확인할 수 있다. 시간과 장소에 구애받지 않고 택배를 보낼 수 있는 편리한 서비스이나 사용자를 잘 고려하지 않은 UI/UX가 아쉽다고 생각되어 리뉴얼 어플로 선정하였다.

UX Vision

앱 사용자 분석 결과, 다수의 이용자가 시간적으로 여유가 부족한 직장인이었고 간편한 절차를 서비스의 강점으로 평가하는 사람이 많았다. user voice에 근거하여 가장 최소한의 조작으로 서비스를 이용할 수 있도록 목표를 설정하였고 사용자 시나리오를 작성하여 주요 화면을 재구성 하였다.

as is
  • 초기 화면에 너무 많은 정보가 나열되 있어 메인과 서브 메뉴를 구분하기 어려움
  • 시각적으로 복잡하고 컨텐츠 영역 사이의 여백이 좁아 전체적으로 가독성이 떨어짐
to be
  • UX  불필요한 카테고리를 간결화하고 최소한의 조작으로 주요한 기능을 사용할 수 있도록 개선
  • UI  직관성이 떨어지거나 터치에서 실수할 수 있는 부분을 사용자의 편의성을 높이는 디자인으로 개선
진행과정
  • 앱 선정
  • 비전과 타겟층 분석
  • IA(Information Achitecture) 분석
  • 경쟁사 앱 분석
  • 비전/목표 설정
  • 페르소나 설정 및 UX Flow 작성
  • 개선점 & 해결책 분석
  • 스타일 키워드 선정 후 가이드 제작
  • 와이어프레임 제작
  • 프로토타입 제작
  • 프로젝트 발표 자료 제작
담당업무
  • 1. 업무 분배, 조원들의 의견 조율 및 반영, 업무 진행 방향 제시
  • 2. 리서치 공동 조사 : 기존 앱, 경쟁사 앱(gs post-box) 사용자 평가 및 장단점 분석
  • 3. ux 비전 설정

작업 평가

  • 잘된 점

    기한 내 작업 수행
    개인의 장점을 파악하여 업무를 분담하였고 작업에 대한 피드백을 활발히하여 보완점을 개선해 나감

    대립을 최소화한 의견 조율
    주관적 의견이 아닌 자료를 토대로 방향을 잡을 수 있도록 고민함

  • 아쉬운 점

    리더의 경험이 없어 강단 있게 의견을 조율하지 못한 부분이 있었으나사사로운 감정에 부딪히지 않기 위해 노력하였고 프로젝트의 목표를 위해 계속해서 소통하였다.

  • 개인적 리뷰

    최선의 해결책을 찾기 위해 몰두하다보니 제작 시간과 수정 시간이 부족하였다. 완성도가 떨어진 부분은 아쉽지만 다양한 개선 방안을 생각하고 구상한 내용을 프로토타입에 적용하는 과정에서 UI와 UX란 어떤 것인지, 사용자 경험 중심으로 사고하고 디자인 하는 것이 무엇인지에 대해 배울 수 있었다.

    소수의 인원이지만 처음으로 프로젝트를 이끌면서 리더의 무게를 느꼈고 좋은 팀워크가 좋은 결과물을 만든다는 것을 알게 되었다. 팀장과 팀원 모두의 입장을 생각할 수 있는 좋은 경험이었고 앞으로 어떤 역할에서도 책임감을 가지고 맡은 바를 잘 해낼 수 있을 것이라고 생각한다.
